# Wallet Setup

## Supported Wallets

Trestle supports all major EVM-compatible wallets through **Reown AppKit**:

- **MetaMask** - Browser extension
- **WalletConnect** - Mobile wallet connection
- **Coinbase Wallet** - Browser and mobile
- **Trust Wallet** - Mobile focused
- **Rainbow** - iOS/Android

## Connecting Your Wallet

1. Click **Connect Wallet** on the Trestle dApp
2. Select your preferred wallet from the modal
3. Approve the connection in your wallet
4. Sign the authentication message

## Network Configuration

Trestle operates on **Polygon Mainnet** and **Amoy Testnet**:

| Network | Chain ID | RPC URL |
|---------|----------|---------|
| Polygon Mainnet | 137 | https://polygon-rpc.com |
| Amoy Testnet | 8002 | https://rpc-amoy.polygon.technology |

## Adding Network to MetaMask

If Polygon is not configured automatically:

1. Open MetaMask → Click network dropdown
2. Select **Add Network** → **Add Network Manually**
3. Enter the RPC details above for your chosen network
4. **Save**

## Security Best Practices

- **Never share** your private keys or seed phrase
- Use a **dedicated wallet** for DeFi interactions
- **Verify contract addresses** on Polygonscan before interacting
- Keep your wallet extension/app **updated**
