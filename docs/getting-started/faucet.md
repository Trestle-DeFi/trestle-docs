# Testnet Faucet

## Getting Started on Testnet

The Trestle Marketplace is currently live on **Polygon Amoy Testnet**. To try it out, you'll need test MATIC tokens.

## Amoy Faucet

Get test MATIC from Polygon's official faucet:

- [Polygon Amoy Faucet](https://faucet.polygon.technology/)

## Alternative Faucets

If the official faucet is unavailable:

- **Alchemy Faucet** - [https://alchemy.com/faucets/polygon-amoy](https://alchemy.com/faucets/polygon-amoy)
- **QuickNode Faucet** - [https://faucet.quicknode.com/drip](https://faucet.quicknode.com/drip)

## Testnet Tokens

Once you have MATIC, you can acquire test tokens:

- **hNOBT** - Available via Telegram Mini-App tasks
