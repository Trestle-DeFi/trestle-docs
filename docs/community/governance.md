# Governance Proposals

## Proposal Process

Trestle Platform governance operates through **on-chain voting** by Governance Token holders.

### Submitting Proposals

Governance Token holders can submit proposals for:

- Platform upgrades
- Marketplace fee structures
- RWA tokenization proposals
- Treasury allocations

### Voting Power

Voting weight is determined by:
- **Governance Token balance**
- **Staking duration** in the Vault
- **Time-weighted** commitment to the ecosystem

## Proposal Lifecycle

```
Draft → Discussion → Vote → Execution → Review
```

### Timeline

| Phase | Duration |
|-------|----------|
| Draft | Flexible |
| Discussion | 7 days minimum |
| Voting | 3 days |
| Execution | After approval |
